Et API som forenkler tittel i iCalendar filer fra TimeEdit, slik at de er leslige.

Kildekoden på GitHub


Ble lei av hvor vanskelig det var å lese iCalendar filer fra HVL sin kalender i TimeEdit, så jeg lagde et API som gjør det enklere. Data for tittelen blir hentet fra den gamle tittelen. For å finne hvilken type hendselse det er, sjekkes beskrivelsen og lokasjonen.

APIet er skrevet i Kotlin med Spring boot, og hosted på en selvhostet tomcat server.
